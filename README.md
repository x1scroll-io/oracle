# x1scroll Oracle

**Decentralized oracle network for X1. First native oracle on any SVM chain.**

Attesters bond 1000+ XNT, submit data, earn fees. False attestations get slashed. 50% of fees go to treasury, 50% burned.

## Feed Types
- ValidatorApy — APY delivered by validator last epoch
- XntPrice — XNT/USD price
- RandomSeed — Verifiable randomness for games/lotteries
- ComplianceCheck — Wallet sanctions check (ZK-lite)
- ValidatorUptime — Validator uptime over N epochs
- Custom — Any data feed
